电脑

快捷键

两个窗口之间互换 使用快捷键【alt+esc】组合快捷键，可以实现在两个窗口之间互换。(注键盘字母左上角) 在多个窗口之间切换 如果想要在在多个窗口之间切换的话，那么可以使用快捷键【alt+tab】。 Ctrl + C：复制选定内容 Ctrl + X：剪切选定内容 Ctrl + V：粘贴内容 Ctrl + A：全选 Ctrl + Z：撤销上一步操作 Ctrl + Y：重做上一步操作 Win + D：显示桌面 Win + M：最小化所有窗口 Win + Shift + M：还原最小化的窗口 文件和应用操作: Ctrl + N：新建窗口或文件 Ctrl + O：打开文件 Ctrl + P：打印文件 Ctrl + S：保存文件 浏览器快捷键: Ctrl + T：新建标签页 Ctrl + W：关闭当前标签页 Alt + F4：关闭当前窗口 Win + R：打开“运行”对话框 其他快捷键 Ctrl + F：查找 Ctrl + H：历史记录 Ctrl + L：在浏览器中聚焦地址栏 回复: F2：文件重命名 F5‌：刷新当前窗口或页面 F11‌：全屏显示（在浏览器中）
